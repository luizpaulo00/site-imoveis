  <action>
1. Install Swiper: `npm install swiper`

2. Create `src/lib/queries/property.ts` with `getPropertyWithImages(id: string)`:
   - Use `createClient` from `@/lib/supabase/server`
   - Fetch single property by id with ALL fields: `select('*, property_images(id, storage_path, is_cover, position)')`
   - Order property_images by position ASC
   - Return null if not found
   - Export return type as `PropertyWithImages`

3. Create `src/components/public/property-gallery.tsx` ('use client'):
   - Import Swiper, SwiperSlide from 'swiper/react'
   - Import modules: Zoom, Navigation, Pagination from 'swiper/modules'
   - Import CSS: 'swiper/css', 'swiper/css/zoom', 'swiper/css/navigation', 'swiper/css/pagination'
   - Accept `images` array (with id, storage_path, is_cover, position) as prop
   - **Inline preview mode:**
     - Swiper with Pagination module, `type: 'fraction'` showing "3/12" format
     - Display images using `getImageUrl(img.storage_path)` -- use the storage path directly (detail variant is same as original in this project since variants weren't implemented as separate paths)
     - Aspect ratio container (16:9 or 4:3), object-cover
     - Click/tap to open fullscreen
     - Track activeIndex with onSlideChange
   - **Fullscreen modal:**
     - Fixed overlay `inset-0 z-50 bg-black`
     - Swiper with Zoom, Navigation, Pagination modules
     - `zoom={true}` for pinch-to-zoom (wraps img in `div.swiper-zoom-container`)
     - Navigation arrows (hidden on mobile via CSS, visible on desktop)
     - Pagination `type: 'fraction'` showing photo count
     - `initialSlide={activeIndex}` to open at current photo
     - Close button (X icon, absolute top-right, white, z-10)
     - Prevent body scroll when fullscreen open
   - Handle empty images: show placeholder
  </action>
